# 📘 Notes

## 简介
> 前人曾照我，我照后来者。  
创建这个项目的初衷是将笔记的格式标准化，并分享给同学们，帮助大家更高效地学习与交流。

## ✨ 项目优点
- **可修改**：欢迎大家积极修改并上传更新的笔记，共同完善内容。  
- **阅读效率高**：相比手写笔记，电子笔记的阅读速度更快。  
- **便于传输**：Markdown 格式体积小，便于快速转发和高效传播。  

## 🛠️ 工具介绍
- **主体部分**：使用 **Markdown** 语法编写，推荐编辑器 **Typora**。  
- **线性规划图解法配图**：可使用 [Graphical Method Calculator](https://www.pmcalculators.com/graphical-method-calculator/) 辅助绘制。  
- **LaTeX 支持**：部分笔记采用 **TeX** 语法，适合数学公式与复杂排版。  

## 📖 使用指南：TeXstudio 与 Overleaf

部分笔记采用 **LaTeX** 文档格式，推荐使用以下工具进行编辑：

### 🖥️ TeXstudio
- **安装**：可在 [TeXstudio 官网](https://www.texstudio.org/) 下载并安装。
- **主要功能**：
  - 语法高亮与自动补全，方便书写复杂公式。
  - 内置 PDF 预览器，可实时查看编译结果。
  - 支持快捷键与宏命令，提高编辑效率。
- **使用步骤**：
  1. 打开 `.tex` 文件。
  2. 点击 “Build & View” 或使用快捷键 `F5` 编译。
  3. 在右侧窗口查看生成的 PDF。

### ☁️ Overleaf
- **访问**：[Overleaf 官网](https://www.overleaf.com/)。
- **主要功能**：
  - 在线协作：多人可同时编辑同一份 LaTeX 文档。
  - 自动编译：无需本地安装环境，直接生成 PDF。
  - 模板丰富：提供大量学术论文、报告模板。
- **使用步骤**：
  1. 登录 Overleaf 并创建新项目。
  2. 上传 `.tex` 文件或复制内容。
  3. 系统会自动编译并生成 PDF。
  4. 可邀请同学协作修改。

📌 **提示**：  
- 如果你只是快速阅读笔记，直接查看生成的 PDF 即可。  
- 如果你想修改或贡献 LaTeX 笔记，推荐使用 **TeXstudio**（本地编辑）或 **Overleaf**（在线协作）。
